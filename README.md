Gestion d’une Agence de Voyage:
pour installer se projet "composer create-project laravel/laravel sarra api"
 run se projet : "php artisan serve"
Introduction:
Ce projet consiste à développer une application web destinée à la gestion d’une agence de voyage. L’objectif principal est de faciliter l’organisation et la gestion des données relatives aux voyages, aux clients et aux réservations.
Objectifs
Gérer les informations des clients
Administrer les voyages proposés par l’agence
Enregistrer et suivre les réservations
Optimiser le processus de gestion interne
Description de l’application
L’application est composée de trois modules principaux :
Clients : ajout, modification et suppression des clients
Voyages : gestion des voyages disponibles
Réservations : enregistrement des réservations en liant les clients aux voyages
Technologies utilisées
Laravel
HTML / CSS / Bootstrap
MySQL
Conclusion
Cette application permet une gestion efficace et structurée d’une agence de voyage, tout en offrant une base évolutive pour des améliorations futures.
